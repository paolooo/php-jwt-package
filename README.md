PHP JWT (JSON Web Token) implementation.

## Install

Via Composer

``` bash
$ composer require league/php-jwt-package
```

## Usage

``` php
$skeleton = new League\Skeleton();
echo $skeleton->echoPhrase('Hello, League!');
```
